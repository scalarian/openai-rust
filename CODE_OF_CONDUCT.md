# Code of Conduct

This project expects direct, respectful, technical collaboration.

## Expected Behavior

- Critique code, docs, tests, and ideas without attacking people.
- Bring evidence when reporting bugs, regressions, or API gaps.
- Keep issues and pull requests scoped to a clear problem.
- Assume good faith while staying precise about technical disagreements.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Bigoted language or exclusionary behavior.
- Spam, trolling, or intentionally disruptive issue and review traffic.
- Publishing private information without permission.

## Scope

This policy applies to repository issues, pull requests, reviews, discussions, and other project-maintained collaboration spaces.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this policy and may restrict repository access when necessary.

For moderator intervention, contact the maintainer at [@staticpayload](https://github.com/staticpayload).
